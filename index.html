Portfolio

This website! I used plain JavaScript so that the navigation menu could be used and display the content accordingly. I also created a version in jQuery to show that I can use that as well. It can be seen on GitHub. The website is just one HTML page, which is why I used JavaScript. I created the Web Designer animation with Illustrator, basing it on a font I liked, then saving it as an SVG and modifying the CSS in the file. It took awhile to create the line, but creating the CSS in the SVG was pretty quick. I've played around with SVG's before and enjoy it. I had the idea to make it look like someone was writing Web Designer on the page and then found some info, with a search, on how to do it.
